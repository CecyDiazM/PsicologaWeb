# Sitio Web Profesional - Psicóloga

https://psicologapamelaarica.com/

Bienvenido al repositorio del sitio web oficial de la **Psicóloga Pamela Camacho Ayca**, una plataforma digital elegante, moderna y optimizada, diseñada para transmitir tranquilidad, profesionalismo y facilitar la gestión de consultas de salud mental en modalidad online.

El diseño del sitio web utiliza una paleta de colores orgánicos y suaves (tonos crema, arena y verde oliva) orientada a la psicología del color para reducir el estrés visual y ofrecer un entorno digital acogedor y empático para los consultantes.

---

## 🌟 Características del Proyecto

* **Interfaz de Usuario Armónica y Relajante:** Diseñada minuciosamente para proyectar paz, equilibrio y cercanía emocional desde el primer contacto.
* **Sección "Sobre mí" y Trayectoria:** Espacio dedicado a la experiencia profesional de la psicóloga en las áreas educacional y clínica.
* **Catálogo de Servicios Detallado:** Desglose interactivo y responsivo de las diferentes modalidades de atención (Consultas, Evaluaciones y Asesorías vía Zoom/Meet).
* **Formulario de Contacto Automatizado (Serverless):** Integración directa con servicios de mensajería externa para recibir las solicitudes directamente en el correo electrónico.
* **Experiencia de Usuario Fluida (UX/UI):** Alertas nativas estilizadas y navegación responsiva adaptada a computadoras y dispositivos móviles.

---

## 🛠️ Tecnologías Utilizadas

Para la construcción de este sitio web se priorizó un desarrollo limpio, eficiente y de alto rendimiento en el lado del cliente:

* **HTML5:** Estructuración semántica y accesible para garantizar un correcto SEO y legibilidad.
* **CSS3 Personalizado:** Uso de Flexbox, CSS Grid, variables dinámicas y transiciones suavizadas para lograr un diseño moderno y orgánico.
* **JavaScript (Vanilla JS):** Lógica nativa para el control de la interactividad del menú colapsable móvil y estados activos de navegación.

---

## 📦 Herramientas y APIs de Terceros (CDNs)

El proyecto prescinde de un backend complejo utilizando integraciones eficientes directas en el navegador:

* **EmailJS (v4):** Permite el procesamiento y envío de los datos del formulario de contacto directamente al correo de la especialista sin necesidad de un servidor propio.
* **SweetAlert2 (v11):** Mejora la experiencia del usuario mostrando ventanas emergentes elegantes y amigables tras el envío exitoso o fallido del mensaje.
* **jQuery (v3.6.1):** Utilizado para simplificar la manipulación del DOM y la integración de eventos del formulario.
* **Bootstrap Icons:** Biblioteca de vectores utilizada para la iconografía del sitio (redes sociales y controles).
* **Google Fonts:** Integración de las familias tipográficas *Playfair Display* (elegancia en títulos) y *Plus Jakarta Sans* (legibilidad y calma en textos corporativos).
